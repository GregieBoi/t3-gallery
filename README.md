# T3 Gallery
T3 Gallery is a simple image gallery built with [T3](https://t3.sh) and [Next.js](https://nextjs.org) to further my learning of Next.js and the included technologies.

Technologies used:
- Uploadthing
- Clerk
- Next.js
- Drizzle ORM
- PostHog
- Upstash
- Tailwind
- Sentry
- Vercel
